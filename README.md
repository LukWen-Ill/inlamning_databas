#Database for Golf Scorecard
Creator: Lukas Wennstrom

Ett digitalt golfscoringssystem som hanterar spelregistrering, scorehantering och statistikgenerering. Databasen är designad som ett OLTP-system (Online Transaction Processing) optimerat för frekventa INSERT/UPDATE-operationer snarare än historisk analys.

Fil Innehåll
schema.sql - CREATE TABLE, datatyper, Keys, Index och vyer
data.sql - Alla INSERT (Data)
queries.sql - Alla frågor enligt kravlistan med kommentar ovanför varje fråga
diagram.png - ER Diagram
dokumentation.pdf - datamodell + motiveringar
